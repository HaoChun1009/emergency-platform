# 緊急事件通報平台

一個讓使用者通報緊急事件的網頁工具。目前為純前端原型版本。

## 功能

- 📋 通報表單:事件類型、緊急程度、地點、描述、通報人、聯絡電話
- 📍 一鍵定位(取得目前經緯度,需 HTTPS 環境)
- ✅ 必填驗證,送出後產生通報編號
- 📱 響應式設計,支援手機

## 檔案結構

| 檔案 | 說明 |
|------|------|
| `index.html` | 頁面結構 |
| `style.css` | 樣式 |
| `app.js` | 互動邏輯(資料目前暫存於瀏覽器 localStorage) |

## 本機預覽

直接用瀏覽器打開 `index.html` 即可。
