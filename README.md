# 緊急事件通報平台

一個 AI 驅動的緊急事件通報網頁工具(Beta 原型)。
使用者透過手機/瀏覽器通報事件,系統自動以 AI 分級並提供處置建議,資料集中於 Google 試算表管理。

> 詳細規劃與進度見 [專案規劃書.md](專案規劃書.md)。

## 功能

- 📋 通報表單:事件類型、緊急程度、地點、描述、通報人、聯絡電話
- 🎤 語音輸入:講話自動填入事件描述
- 📍 一鍵定位:取得 GPS 並自動轉成地址(需 HTTPS)
- 🧩 動態欄位:依事件類型顯示對應追問(如火災→是否受困)
- 📷 現場照片:可選相簿或拍照,前端自動壓縮後上傳
- 🤖 AI 智能分級:Gemini 結合文字+照片,依 SOP 判三級(重大災害/廠級事故/一般事故)並給處置建議
- 📊 集中管理:通報寫入 Google 試算表,照片存雲端硬碟

## 架構

```
前端(GitHub Pages 靜態網頁)
   └─→ Google Apps Script 後端
          ├─ Gemini API:分級 + 處置建議(含照片辨識)
          ├─ RAG:讀取試算表「SOP知識庫」分頁
          ├─ Google 試算表:資料儲存 / 管理後台
          └─ Google 雲端硬碟:照片儲存
```

## 檔案結構

| `專案規劃書.md` | 完整規劃與進度 |

## 安全說明

- AI 金鑰存於後端(Apps Script 指令碼屬性),不在前端程式碼中。
- SOP 內容與個資**不放入此公開倉庫**,改存於非公開的 Google 試算表。
